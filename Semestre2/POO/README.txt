# Programación Orientada a Objetos (Semestre 2)

En esta materia aprendí los pilares fundamentales del paradigma orientado a objetos y cómo estructurar software de forma modular y escalable.

## Conceptos Clave Dominados
* **Abstracción y Encapsulamiento:** Diseño de clases con atributos privados y métodos públicos.
* **Herencia y Polimorfismo:** Creación de jerarquías de clases para reutilizar código y comportamiento dinámico.
* **Modelado:** Creación de diagramas de clases (UML) para la planeación del software.

## Estructura de las Carpetas
* **Unidades 1 a 6:** Prácticas de clase, ensayos de conceptos de conceptos.
* **Proyecto Final:** Un programa interactivo que simula el menú de una veterinaria, se utilizan interfaces, polimorfismo, herencia, enums, etc.

## Diseño y Arquitectura Destacada

Durante este semestre, adelanté el desarrollo del proyecto final, lo que me permitió comprender a fondo la estructura del sistema antes de finalizar las entregas teóricas. Como resultado, diseñé un modelo UML robusto y detallado para una de las entregas principales (Unidad4 > Unidad4ExamenPractico), aplicando de forma óptima los principios de la programación orientada a objetos.
